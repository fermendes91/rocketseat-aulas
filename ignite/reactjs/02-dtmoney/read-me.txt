Para criar este projeto foi utilizado o comando com

yarn create react-app --template typescript

desta forma o projeto já criado pronto para utilizar o typescript.

Também foi instalado o styled-components para melhorar a forma como trabalhar com estilos dentro do projeto.

yarn add styled-components 
yarn add @types/styled-components -D


Para simular e preparar o nosso ambiente para chamada de API's podemos utilizar o miragejs para simular
na configuração no projeto

o /api é o caminho padrão da API não se altera. o this.get() é o mock de serviço para a requisição.

yarn add miragejs
